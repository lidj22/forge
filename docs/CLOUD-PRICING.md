# 云服务器部署成本

---

## 你的需求

- 5-10 个并发 Agent 会话（调用 AI API，不跑本地模型）
- Node.js / Python 后端 + SQLite + Web Dashboard
- 7×24 在线

**这是一个轻量 I/O 密集型负载**，不需要 GPU，4-8GB RAM 足够。

---

## 价格对比

### 最佳性价比

| 方案 | 配置 | 月费 | 评价 |
|------|------|------|------|
| **Oracle Cloud Free** | 4 核 ARM, 24GB RAM, 200GB | **$0** | 业界最强免费，但实例抢不到且可能被回收 |
| **Hetzner CX32** | 4 vCPU, 8GB, 80GB SSD | **~$7.5/月** | **最推荐**，欧洲/美国机房，性价比之王 |
| **腾讯云 (年付)** | 4C/8G/12M | **~¥36/月 (年付¥435)** | 国内最便宜，但需备案，AI API 延迟高 |
| **阿里云 (年付)** | 4C/8G | **~¥80/月 (年付¥955)** | 类似腾讯云，香港节点可避免备案 |

## 完整费用估算（月）

| 项目 | 费用 |
|------|------|
| 服务器 (Hetzner CX32) | ~$7.5 |

| **合计** | **~$8.5/月 (~¥62)** |

加上 AI API 费用：
| AI Provider | 费用 |
|-------------|------|
| Gemini Free | $0 |
| Grok Free | $0 |
| Claude API | 按用量，~$10-50/月 |
| OpenAI (如需) | 按用量 |
| **AI 合计** | **$0-50/月**（取决于使用量）|

**总计：服务器 + AI = ~$10-60/月**
